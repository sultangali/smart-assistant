# Руководство по миграции: Многоязычные описания и новые категории

## Обзор изменений

Данное обновление включает в себя следующие изменения:

### 1. Обновленные категории и подкатегории

Добавлены 7 новых категорий с подкатегориями:

1. **Персонализированное обучение**
   - Адаптивное обучение
   - Образовательная аналитика

2. **Интерактивные и опытно-ориентированные технологии**
   - Игровые и иммерсивные технологии
   - Виртуальные лаборатории и симуляции

3. **Технологические платформы и инструменты для разработки образовательных решений**
   - Платформы для создания контента
   - Образовательные среды и LMS интеграции

4. **Оценка и обратная связь**
   - Автоматизированное оценивание
   - Таблицы и аналитика успеваемости

5. **Инклюзия и доступность**
   - Поддержка доступного обучения
   - Инклюзивные технологии

6. **Создание образовательного контента**
   - Генерация текстовой информации
   - Создание аудио и видео контента

7. **Поддержка обучения (виртуальные ассистенты и платформы)**
   - Виртуальные ассистенты
   - Платформы управления образовательной средой

### 2. Многоязычные описания инструментов

Модель `Tool` теперь поддерживает описания на трех языках:
- Русский (ru)
- Английский (en)
- Казахский (kk)

Описание автоматически отображается на языке интерфейса пользователя.

### 3. Обновленная админ-панель

Модальное окно добавления/редактирования инструмента теперь включает:
- Поля для ввода описания на трех языках
- Переведенные категории и подкатегории
- Улучшенный пользовательский интерфейс

## Шаги миграции

### Шаг 1: Обновление зависимостей

```bash
cd client
npm install

cd ../server
npm install
```

### Шаг 2: Миграция базы данных

Запустите скрипт миграции для обновления существующих инструментов:

```bash
cd server
node scripts/migrateToolDescriptions.js
```

Этот скрипт:
- Добавит поле `descriptions` ко всем существующим инструментам
- Скопирует текущее описание во все языковые версии
- Оставит существующее поле `description` для обратной совместимости

### Шаг 3: Перезапуск приложения

```bash
# Терминал 1 - Сервер
cd server
npm start

# Терминал 2 - Клиент
cd client
npm run dev
```

## Обновление существующих инструментов

После миграции рекомендуется:

1. Войти в админ-панель
2. Отредактировать каждый инструмент
3. Добавить корректные переводы описаний на английский и казахский языки
4. Обновить категории инструментов в соответствии с новой иерархией

## Структура данных

### До миграции:
```javascript
{
  title: "Tool Name",
  description: "Tool description",
  category: "Old Category",
  subcategory: "Old Subcategory"
}
```

### После миграции:
```javascript
{
  title: "Tool Name",
  description: "Tool description", // сохраняется для совместимости
  descriptions: {
    ru: "Описание на русском",
    en: "Description in English",
    kk: "Қазақша сипаттама"
  },
  category: "Персонализированное обучение",
  subcategory: "Адаптивное обучение"
}
```

## API изменения

### Создание/обновление инструмента

**POST** `/api/tools` или **PUT** `/api/tools/:id`

```json
{
  "title": "ChatGPT",
  "description": "AI-powered chatbot",
  "descriptions": {
    "ru": "ИИ-чат-бот для различных задач",
    "en": "AI-powered chatbot for various tasks",
    "kk": "Әртүрлі тапсырмаларға арналған ЖИ-чат бот"
  },
  "category": "Поддержка обучения (виртуальные ассистенты и платформы)",
  "subcategory": "Виртуальные ассистенты",
  "functions": ["Генерация текста", "Ответы на вопросы"],
  "purpose": "Помощник в обучении",
  "link": "https://chat.openai.com",
  "color": "#10A37F"

```

## Поддержка

При возникновении проблем проверьте:
- Логи сервера
- Консоль браузера
- Соединение с базой данных
- Правильность переменных окружения в `.env`

## Дополнительные возможности

### Добавление нового языка

1. Добавьте новый файл перевода: `client/src/i18n/locales/{lang}.json`
2. Обновите модель Tool: добавьте новое поле в `descriptions`
3. Обновите компоненты для поддержки нового языка
4. Добавьте опцию в LanguageSwitcher
